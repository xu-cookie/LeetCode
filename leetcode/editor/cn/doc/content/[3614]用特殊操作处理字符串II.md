# [3614] 用特殊操作处理字符串 II

**难度**: Hard  
**题目链接**: https://leetcode.cn/problems/process-string-with-special-operations-ii/

## 题目描述

给定一个由小写英文字母和特殊字符 `'*'`、`'#'`、`'%'` 组成的字符串 `s`，以及一个整数 `k`。

按照以下规则从左到右处理 `s` 构建新字符串 `result`：

1. 如果字符是**小写英文字母**，将其追加到 `result` 末尾
2. `'*'` **删除** `result` 的最后一个字符（如果存在）
3. `'#'` **复制**当前 `result` 并追加到自身末尾（即 `result = result + result`）
4. `'%'` **反转**当前 `result`

返回最终 `result` 的第 `k` 个字符（**0-indexed**）。如果 `k` 超出 `result` 的范围，返回 `'.'`。

## 示例

### 示例 1
```
输入: s = "a#b%*", k = 1
输出: "a"
```

| i | s[i] | 操作 | 当前 result |
|---|------|------|-------------|
| 0 | 'a' | 追加 'a' | "a" |
| 1 | '#' | 复制 result | "aa" |
| 2 | 'b' | 追加 'b' | "aab" |
| 3 | '%' | 反转 result | "baa" |
| 4 | '*' | 删除最后一个字符 | "ba" |

最终 `result = "ba"`，第 `k = 1` 个字符是 `'a'`。

### 示例 2
```
输入: s = "cd%#*#", k = 3
输出: "d"
```
最终 `result = "dcddcd"`，第 `k = 3` 个字符是 `'d'`。

### 示例 3
```
输入: s = "z*#", k = 0
输出: "."
```
最终 `result = ""`，`k = 0` 超出范围，返回 `'.'`。

## 约束

- `1 <= s.length <= 10^5`
- `s` 仅包含小写英文字母和特殊字符 `'*'`, `'#'`, `'%'`
- `0 <= k <= 10^15`
- 处理 `s` 后的 `result` 长度不超过 `10^15`

## 解题思路

### 核心方法：逆向追踪

由于 `result` 最终长度可达 `10^15`，直接模拟构建字符串不可行。采用**两遍遍历+逆向追踪**：

#### 第一遍（正向）：计算长度
- 遍历 `s`，记录每次操作后 `result` 的长度（用 `long[]` 存储）
- 字母：`len++`
- `*`：`len = max(0, len-1)`
- `#`：`len *= 2`
- `%`：长度不变
- 长度计算不会溢出 `long`（最大仅 `10^15`）

#### 第二遍（逆向）：追踪字符来源
从最后一个操作向前遍历，维护当前关注的索引 `k`：

- **字母**：该字母被追加在位置 `curLen - 1`。如果 `k == curLen - 1`，直接返回该字母；否则 `k` 不变
- **`*`**（逆向）：正向删除了末尾字符，逆向恢复时 `k` 位置不变（被删字符在末尾之后）
- **`%`**（逆向）：反转的逆还是反转，`k = curLen - 1 - k`
- **`#`**（逆向）：复制操作的逆是取模，`k = k % (curLen/2)`

### 复杂度分析
- **时间复杂度**: O(n)，两次遍历字符串
- **空间复杂度**: O(n)，存储长度数组

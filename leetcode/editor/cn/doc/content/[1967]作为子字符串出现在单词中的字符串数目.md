# [1967] 作为子字符串出现在单词中的字符串数目

**难度:** 简单 (Easy)
**链接:** https://leetcode.cn/problems/number-of-strings-that-appear-as-substrings-in-word/

## 题目描述

给你一个字符串数组 `patterns` 和一个字符串 `word`，统计 `patterns` 中有多少个字符串是 `word` 的**子字符串**。返回字符串数目。

**子字符串**是字符串中的一个连续字符序列。

## 示例

**示例 1:** `patterns = ["a","abc","bc","d"]`, `word = "abc"` → `3`

**示例 2:** `patterns = ["a","b","c"]`, `word = "aaaaabbbbb"` → `2`

**示例 3:** `patterns = ["a","a","a"]`, `word = "ab"` → `3`

## 约束

- `1 <= patterns.length <= 100`
- `1 <= patterns[i].length <= 100`
- `1 <= word.length <= 100`
- `patterns[i]` 和 `word` 由小写英文字母组成

## 解题思路

遍历 `patterns` 数组，使用 Java 内置的 `String.contains()` 方法判断每个 pattern 是否为 word 的子串，统计计数。

## 复杂度

- **时间复杂度:** O(n * m)
- **空间复杂度:** O(1)

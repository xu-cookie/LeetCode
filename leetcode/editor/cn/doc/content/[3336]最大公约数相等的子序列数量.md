# [3336] 最大公约数相等的子序列数量

- 难度: Hard
- 题目: https://leetcode.cn/problems/find-the-number-of-subsequences-with-equal-gcd/

## 题目描述

给你一个整数数组 `nums`。

你的任务是找出 `nums` 中满足以下条件的 **非空** 子序列对 `(seq1, seq2)` 的数量：

- `seq1` 和 `seq2` 是 **不相交** 的，即 `nums` 中的 **同一个下标** 不会同时出现在两者中。
- `seq1` 元素的 GCD 等于 `seq2` 元素的 GCD。

返回满足条件的子序列对的总数，结果对 `10^9 + 7` 取余。

## 示例

**示例 1:**
- 输入: `nums = [1,2,3,4]`
- 输出: `10`

**示例 2:**
- 输入: `nums = [10,20,30]`
- 输出: `2`

**示例 3:**
- 输入: `nums = [1,1,1,1]`
- 输出: `50`

## 约束

- `1 <= nums.length <= 200`
- `1 <= nums[i] <= 200`
